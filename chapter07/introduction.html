7.1. Вступление

Эта глава описывает конфигурационные файлы и загрузочные скрипты. Сперва создаются основные файлы конфигурации, необходимые для настройки сети.

  • Раздел 7.2, 첝eneral Network Configuration.

  • Раздел 7.3, Создание файла /etc/hosts.

Затем обсуждаются проблемы правильной настройки устройств.

  • Раздел 7.4, 첗evice and Module Handling on an LFS System.

  • Раздел 7.5, Создание собственных ссылок на устройства.

Последующие разделы описывают установку и настройку загрузочных скриптов LFS, необходимых для инициализации системы при включении. Большая часть этих скриптов будет работать без изменений, но все же необходимо создать несколько дополнительных конфигурационных файлов, поскольку они будут предоставлять информацию об оборудовании.

Загрузочные скрипты в стиле System V выбраны в этой книге по причине их наибольшей распространенности и относительной простоты. Совет, описывающий установку загрузочных скриптов в стиле BSD, можно прочесть здесь: http://www.linuxfromscratch.org/hints/downloads/files/bsd-init.txt. Поиск в списках рассылки LFS по словам depinit, upstart и systemd позволит Вам ознакомиться с дополнительными вариантами.

Если Вы используете альтернативный стиль загрузочных скриптов, пропустите эти разделы.
